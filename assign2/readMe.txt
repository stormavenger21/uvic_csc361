##CSC 361: A2 TCP Traffic Analysis Documentation

##required files. Must all be in same directory:
packet_struct.py
tcp.py
<name of file>.cap

## Running tcp.py
1. Install Python 3.8.10 or higher
2. Open Terminal on UNIX based OS
3. Navigate to the directory containing tcp.py AND packet_struct.py AND the .cap file
4. Execute the following command in Terminal:

    python3 tcp.py <cap file name>
	

#Examples:
python3 tcp.py "tcpdump1_reth0.cap"
python3 tcp.py "sample-capture-file.cap"
